Deprecation Policy¶

As of release 1.3.0, Envoy will follow a Breaking Change Policy.

Features in the deprecated list for each version have been DEPRECATED and will be removed in the specified release cycle. A logged warning is expected for each deprecated item that is in deprecation window.
